# 😂 MemeFactory

> **表情包工厂 — 一键生成你的专属表情包**

选模板 → 加文字 → 导出PNG，三步搞定表情包

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🎭 **24个内置模板** | 经典/工作/心情/社交 4大分类 |
| 📸 **自定义上传** | 上传任意图片做表情包 |
| ✏️ **经典Meme文字** | Impact字体 + 黑色描边，正宗Meme味 |
| 🎯 **快捷文案** | 一键填入经典Meme配文 |
| 💾 **自动保存** | 导出后自动保存到画廊 |
| 📥 **PNG导出** | html2canvas 高清2x导出 |
| 🌙 **暗色主题** | 护眼创作 |

---

## 🎮 使用流程

```
1. 选模板 → 经典/工作/心情/社交 24款
2. 编辑文字 → 上方/下方文字 + 字体大小
3. 导出保存 → 一键下载PNG + 自动保存画廊
```

也支持上传自己的图片！

## 🚀 快速开始

```bash
git clone https://github.com/PHclaw/meme-factory.git
cd meme-factory
npm install
npm run dev
```

打开 http://localhost:5173 开始造表情包。
